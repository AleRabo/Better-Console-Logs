# Better-Console-Logs

More logs for youre console, Require [Exiled v2.10.0](https://github.com/Exiled-Team/EXILED/releases/tag/2.10.0-beta)


Config:
```
better_console_logs:
  is_enabled: true
  
  #are RA command logs enabled?
  command_logs: false
  ```
